# TUI 使用说明

## 启动

使用 Go 1.26+ 构建后，可通过以下命令进入 TUI：

```bash
coding-agent tui
```

## 快捷键

- `Enter`: 发送输入消息
- `Esc`: 中断当前轮输出（不中断会话）
- `Ctrl+C`: 退出 TUI
- `Up/Down` 或 `j/k`: 滚动消息视图

## 当前能力范围

- 并行于 `chat`/`once` 的新入口，不替换原有命令
- 消息流展示与输入区
- Agent 会话对接与增量流式渲染
- 错误提示可视化与中断反馈

## 已知限制

- 当前会话环境若低于 Go 1.26，可能无法完成全仓构建验证
- TUI 下需要交互审批的高风险工具调用当前默认拒绝（避免 raw stdin 询问导致阻塞）
- 首版不包含会话管理面板、配置面板、多窗口布局
- 主题与鼠标交互仅提供基础能力

## 后续扩展方向

- 会话列表与恢复的可视化管理
- 任务/子代理状态看板
- 更完整的主题与布局配置
